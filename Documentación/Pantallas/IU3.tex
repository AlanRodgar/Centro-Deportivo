\subsection{IU3 Pantalla de Registrar Cliente}

\subsubsection{Objetivo}
	Ingresar los datos de un cliente al sistema para que se almacene su información.

\subsubsection{Diseño}
Esta pantalla aparece al seleccionar la opción ``Registrar'' de la \IUref{IU1}{Página principal.} Se compone de un formulario con campos de texto para que se digite la información.

Todos los campos marcados con * son obligatorios.

\IUfig[.6]{gui/formClientes}{IU3}{Pantalla de Registrar Clientes.}

\subsubsection{Comandos}
\begin{itemize}
	\item \IUbutton{Guardar}: Verifica que el CURP del cliente no se encuentre registrado previamente y que todos los campos hayan sido llenados de forma correcta. Si la verificación es correcta, se muestra el mensaje {\bf MSG16-}``Registro exitoso. Tú número de cliente es: [{\em Número de cliente}]'' y se redirecciona a la \IUref{IU4}{Venta de Membresía.} 
\end{itemize}

\begin{itemize}
	\item \IUbutton{Reset}: Limpia todos los campos de texto que hayan sido rellenados con alguna información para que se vuelvan a introducir los datos.
\end{itemize}
